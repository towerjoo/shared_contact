About this script
======================

You can run `shared_contacts_profiles.py help` to get help directly.

And you can follow the steps below to start:

1. copy the *fields_mapping.default.py* to *fields_mapping.py*
2. edit the *fields_mapping.py* to fit your csvs to import
3. run `shared_contacts_profiles.py --import=INFILE` to do the importing
